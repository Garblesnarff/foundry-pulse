# Foundry Pulse — App Store Metadata

## Title
Foundry Pulse — Mac Hardware Monitor

## Subtitle
Read the Heat of Your Mac. All Local.

## Description

100% local. 100% offline. 100% yours.

Your Mac runs hot. Foundry Pulse watches the temperature of every circuit — CPU, GPU, memory, disk, and battery — surfacing live metrics in your menu bar before thermal throttling costs you time. Built specifically for Apple Silicon.

**REAL-TIME METRICS**
• CPU usage (total + per-core breakdown)
• GPU usage and temperature
• Memory: used, compressed, swap
• Disk: read/write throughput and free space
• Network: upload/download per interface
• Battery: charge level, cycle count, health
• Temperature sensors: CPU die, GPU, storage
• Fan speed and thermal throttling state
• Top CPU and RAM consuming processes

**MENU BAR**
• Live sparkline graph in menu bar
• Color-coded by utilization (green → yellow → red)
• Click to open full popover with detailed metrics

**HISTORICAL GRAPHS**
• 1-minute, 1-hour rolling charts
• 24-hour history (Pro)
• SwiftCharts-based smooth curves
• Tap to zoom and inspect

**SMART ALERTS (Pro)**
• Configurable thresholds: CPU > 80%, RAM > 90%
• Thermal throttling detection
• Battery health degradation warning
• 5-second check interval, duplicate suppression

**WIDGETS (Pro)**
• 2×2 widget: CPU gauge with mini graph
• 3×3 widget: 4-quadrant CPU/GPU/RAM/Disk

Pro IAP unlocks configurable alerts, macOS widgets, and 24-hour history.
